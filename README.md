# 🌐 Умный VPN для дома

Набор правил для роутера с **v2rayA**, который сам понимает куда идти напрямую, а куда через VPN.

## 🎯 Как это работает

Всё что связано с Россией, играми и обновлениями — работает напрямую на полной скорости. Зарубежные сайты, Google, YouTube — идут через VPN в Нидерландах.

> Никаких кнопок «включить/выключить» — всё работает само.

## 📦 Что внутри

### 🇷🇺 Россия напрямую
Госуслуги, банки, ВКонтакте, Mail.ru, Яндекс, ОК, Дзен, Авито, Ozon, Wildberries — весь `.ru` интернет летит мимо VPN

Steam, Epic Games, Ubisoft Connect, EA, Rockstar, Battle.net, Riot, War Thunder, Genshin Impact, Mobile Legends, Wargaming, Gaijin

### 🇨🇳 Китайские сервисы напрямую

### 🐧 Linux напрямую
Arch Linux, AUR, CachyOS — обновления системы качаются без прокси

### 🪟 Windows напрямую
Обновления Windows Update, проверки подключения к интернету, синхронизация времени

### 🌍 Остальное через VPN
Google, YouTube, Gmail, GitHub, Discord, Reddit, Twitter, OpenAI и весь остальной мир — надёжно через Нидерланды

## 🔧 Как установить

1. Открой веб-интерфейс роутера с v2rayA
2. Перейди в раздел **Маршрутизация RoutingA**
3. Скопируй содержимое файла [`routingA.conf`](routingA.conf)
4. Нажми **Сохранить**

Готово. Роутер сам разберётся куда идти напрямую, а куда через VPN.

## 📝 Настройки v2rayA

- Режим: **Прозрачный прокси → Redirect**
- Разделение трафика: **RoutingA** (режим с правилами)
- Порт: по умолчанию
